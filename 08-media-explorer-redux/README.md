📸 Media Explorer (React + Redux Toolkit)

A media search application built using React, Redux Toolkit, Unsplash, and Pexels APIs.
Users can search for photos or videos, switch between media types, and save their favorites to a personal collection stored in localStorage. Built as a learning project to explore API integration, Redux Toolkit state management, and UI interactions.

🚀 Features:
🔍 Search Photos (Unsplash API)
🎥 Search Videos (Pexels API)
🗂️ Save items to Collections (localStorage)
🔄 Switch between Photos / Videos tab
⚛️ State management using Redux Toolkit
🔔 Toast notifications (React-Toastify)
⚡ Built with Vite for lightning-fast dev
📁 Modular JavaScript utilities for cleaner logic
